Look ! This is Milky

A front-end boilerplate for modern projects.

Download View on Github

Milky is an opinionated boilerplate built for the modern web developer. It is based on the latest technologies and will help you improve your workflow. It comes packed with some pretty neat features and a build process to avoid repetitive tasks so you can focus on your code.

Milky is Html5 ready, which means you can start creating fully semantic, accessible markup for your websites

Milky is SASS powered, using indented syntax. Sorry, no Scss ! It also comes with Bourbon and the Neat grid system.

Insant browser refresh, Jade and Sass compiling, Image, JS/CSS minification and autoprefixing are all taken care of by this super fast Build tool.
